日期函数

  • Date() 返回当日的日期和时间。
  • Date.getDate(): number Gets the day-of-the-month(1 ~ 31), using local time.
  • Date.getDay(): number Gets the day of the week(0 ~ 6), using local time.
  • Date.getMonth(): number Gets the month (0 ~ 11), using local time.
  • Date.getFullYear(): number Gets the year, using local time.
  • Date.getHours(): number Gets the hours in a date, using local time.
  • Date.getMinutes(): number Gets the minutes of a Date object, using local time.
  • Date.getSeconds(): number Gets the seconds of a Date object, using local time.
  • Date.getMilliseconds(): number Gets the milliseconds of a Date, using local time.
  • Date.getTimezoneOffset(): number Gets the difference in minutes between the time on the local computer and Universal Coordinated Time (UTC).
  • Date.getUTCDate(): number Gets the day-of-the-month, using Universal Coordinated Time (UTC).
  • Date.getUTCHours(): number Gets the hours value in a Date object using Universal Coordinated Time (UTC).
  • Date.toString(): string Returns a string representation of a date. The format of the string depends on the locale.
  • Date.toDateString(): string Returns a date as a string value.
  • Date.toTimeString(): string Returns a time as a string value.
  • Date.toUTCString(): string Returns a date converted to a string using Universal Coordinated Time (UTC).
  • Date.toLocaleString( locales?: string | string[], options?: Intl.DateTimeFormatOptions): string Converts a date and time to a string by using the current or specified locale.
  • Date.toLocaleDateString( locales?: string | string[], options?: Intl.DateTimeFormatOptions): string Converts a date to a string by using the current or specified locale.
  • Date.toLocaleTimeString( locales?: string | string[], options?: Intl.DateTimeFormatOptions): string Converts a time to a string by using the current or specified locale.
